# Product Web

Vite React reference UI for the local product slice.

## Run

From the repository root:

```bash
pnpm dev:web
```

Or from this directory:

```bash
pnpm dev
```

The dev server listens on `http://localhost:5173`.

## Environment

The UI defaults to `http://localhost:8000` for API calls. Override with:

```bash
VITE_REFERENCE_API_BASE_URL=http://localhost:8000
```

## Checks

```bash
pnpm test
```
